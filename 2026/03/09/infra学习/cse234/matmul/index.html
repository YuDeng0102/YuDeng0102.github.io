GPU Matmul

这节课从最简单的matmul开始，然后按照tiling,分块，shared memory等优化策略，实现一个高效的matmul算子

GPU Matmul v1

先从最简单的矩阵乘法开始，每个线程计算C矩阵中的一个元素

GPU Matmul v1.5 Thread Tiling


让每个线程计算一个 大小的子矩阵的元素，Global Memory的访问次数从 减少到了 ,原因是对A矩阵的每行，同时计算了列B矩阵的元素

GPU Matmul v2 分块


原本每个线程计算A的一行需要访问B的V列，这样会导致B的每一列被重复访问了V次，因为A的变化幅度比B慢。可以考虑分块，让对A,B的访问同步进行，这样避免了B的每一列被重复访问。

GPU Matmul v3 SRAM Tiling


利用shared memory,每个block计算一个 的子矩阵,每个线程计算一个 大小的子矩阵的元素，通过共享内存减少了对Global Memory的访问次数，此外，这里每个block在计算时加载Global Memory是 cooperative fetch的，如下：
